# @reley/shared

Shared types + zod schemas + IPC method names + error envelope for the Reley toolchain.

Imported by `@reley/core`, `@reley/core-cli`, the Reley Desktop app, and Reley Cloud — no runtime engine, no DOM, no Node-only deps.

## Install

```sh
npm i @reley/shared zod
```

Full docs: https://github.com/relay-protocol/reley
